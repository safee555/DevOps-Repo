What is DevOps?

DevOps is a cultural and technical movement focused on unifying development and operations into a single, automated, and collaborative process. This website gives you everything: tools, concepts, pipelines, visual examples, and real-world workflows.

Configuration management and automation tool.

Infrastructure as Code tool for provisioning cloud resources.
